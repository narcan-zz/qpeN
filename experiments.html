Experiments
 
E5calcMC - A program written to re-familiarize myself with Java's syntax and try to format an ASCII message into a number that can be processed by the encryption equation.
E7calcPQ - A program written to develop an algorithm to produce prime numbers that are "random enough" for the RSA encryption algorithm.
E8calcD - A program used to verify the equation used to derive the private key 'd' from e and the phi function.
E10calcCM - A program used to ensure that the deciphering equation works in the context that I need, and to explore the methods of translating the deciphered plaintext.
E11transM - A program used to test an algorithm for translating the deciphered plaintext. It was abandoned before completion due to other discoveries, but its principles are used in the operational version of the qpeN™ translation.
E12MCM - A program consolidated from other experiments to convert a text message into a number that can be processed by the encryption equation and use that encrypted message to derive the original number from the decryption equation.
E13transM - A program written to show yet another algorithm for translating a number deciphered from the decryption equation into a text message.
modTest - A program written to show how to add the signing and verification option to the application.
EqpeN - A working demo version of qpeN™ using many of the principles from the above experiments to show all of the working parts that will go into the final qpeN™ product. Fully functional for demonstration purposes. Requires a test file to encipher and decipher called "test.txt"
